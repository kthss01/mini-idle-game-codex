# PR3 성장/경제 밸런스 설계

## 성장 곡선 선택
- 비용 곡선은 **지수형**을 사용합니다.
- 공격력 강화 비용: `cost = floor(10 * 1.18^level)`
- 체력 강화 비용: `cost = floor(12 * 1.16^level)`
- 강화 1회당 스탯 증가:
  - 공격력: 현재 공격력의 `+8%` (내림)
  - 체력: 현재 최대 체력의 `+10%` (내림)

## 선택 근거
- Idle 장르에서 강화 효율 체감이 빠르게 오고, 이후 투자 선택이 필요해지는 흐름을 만들기 위해 지수형 비용을 선택했습니다.
- 공격력은 처치 속도(TTK)에 직접 반영되고, 체력은 생존 시간과 연속 파밍 안정성을 높이는 역할로 분리했습니다.
- 공격력/체력의 비용 증가율을 다르게 둬서 특정 스탯 몰빵이 항상 정답이 되지 않도록 조정했습니다.
