# PR3 성장/경제 밸런스 설계

## 선택한 성장 곡선
- 공격력 강화 비용: `10 * 1.18^level` (내림)
- 체력 강화 비용: `12 * 1.16^level` (내림)
- 공격력 강화 효과: 현재 공격력의 `+8%`
- 체력 강화 효과: 현재 최대 체력의 `+10%`

지수 비용 곡선을 선택한 이유:
1. 초반에는 낮은 비용으로 빠른 체감(5~10분 내 성장 경험)을 제공.
2. 중반부터 비용이 빠르게 상승해 골드 소비처 역할을 하며 인플레이션 완화.
3. 공격/체력의 비용 증가율을 다르게 둬 한쪽 몰빵의 효율을 자연스럽게 제한.

## 체감 지표 점검 기준
- DPS는 `atk / (cooldownMs / 1000)`로 계산.
- 생존력은 `hp / (monsterDps)`로 계산.
- 강화 직후 UI에서 DPS/예상 생존시간이 즉시 갱신되어 체감 가능.

## 밸런스 가이드
- 공격만 올리면 처치 속도는 빨라지지만 사망 리스크가 커지므로 체력 강화 필요.
- 체력만 올리면 생존은 늘지만 처치가 느려져 골드 파밍 효율 저하.
- 결과적으로 두 강화의 교차 투자가 가장 효율적인 구조를 의도.
