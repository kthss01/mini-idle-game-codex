# PR3 성장 + 경제 설계 노트

## 성장 곡선 선택
- 비용 곡선은 **지수형**을 채택했다.
- 공격력 강화 비용: `cost = floor(10 * 1.18^level)`
- 체력 강화 비용: `cost = floor(12 * 1.16^level)`
- 업그레이드 수치:
  - 공격력 강화 1회: 공격력 `+8%` (내림)
  - 체력 강화 1회: 최대 체력 `+10%` (내림), 증가분만큼 즉시 회복

## 선택 근거
- Idle 게임의 핵심 체감은 "초반 빠른 구매 → 중반 선택 압박"이다.
- 공격력 강화를 조금 더 가파른 비용 곡선(1.18)으로 두어 DPS 몰빵을 억제했다.
- 체력 강화는 1.16으로 완만하게 두어 안정성을 위한 선택지를 유지했다.
- 두 스탯 모두 퍼센트 증가를 채택해 초반/중반/후반에서 성장 체감이 끊기지 않도록 했다.

## 플레이 체감 검증 포인트
- 초반(5~10분): Lv1~8 구간에서 두 강화 모두 짧은 주기로 구매 가능.
- 중반: 공격력 비용이 빠르게 상승하므로 체력 투자와 분산 투자 유도.
- 회귀 위험: 공격력/체력 어느 한쪽만 몰빵해도 반대 축(생존 시간, 처치 속도)이 즉시 병목이 되도록 설계.
