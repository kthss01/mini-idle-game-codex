# 작업 내역 정리

## 개요
최근 변경 이력을 빠르게 파악할 수 있도록 주요 커밋을 한국어 기준으로 정리했습니다.

## 최근 작업 요약
- PR3(성장/업그레이드) 관련 병합 충돌을 단계적으로 정리해 UI/로직 회귀를 복구했습니다.
- 충돌 마커 검사 스크립트의 오탐 이슈를 수정했습니다.
- 문서 운영 원칙(문서 동시 업데이트, 한국어 작성 기준)을 명문화했습니다.
