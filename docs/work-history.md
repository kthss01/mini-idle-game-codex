# 작업 내역 정리

## 개요
지금까지 진행된 작업 흐름을 한눈에 파악할 수 있도록 주요 변경 사항과 커밋 내역을
요약했습니다.

## 주요 작업 요약
- 전투 MVP 씬과 전투 루프/상태 초기화 로직을 정리하며 충돌을 해결했습니다.
- 전투 스케일링과 밸런스 데이터를 적용해 전투 상태를 보정했습니다.
- 몬스터/장비 데이터 및 성장 곡선 정의를 추가하면서 데이터 기반 확장 기반을 마련했습니다.
